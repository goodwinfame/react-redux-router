# react-redux-router
react seed
===========
<p>这是一个react demo，可以作为seed使用，方便自己快速搭建react项目</p>
<p>包含react + redux + router + ant.design</p>
### 安装项目
<pre><code>npm install</code></pre>
### 开启监听
<pre><code>npm run watch</code></pre> 
<p>浏览器访问：</p>
<code>http://localhost:63340/webpack-dev-server/</code>
